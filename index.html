Progetto di informatica

Presentazione Progetto:

Il sito Web è stato completato in 2 mesi. Si basa su 3 sistemi di programmazione (Html, Css, Java Script) con un'aggiunta di animazioni per non rendderlo troppo statico. Per questo sito ho puntato molto sulla grafica e sul contenuto. Ho scelto di parlare della scuola e sull'indirizzo che frequento. Sulla scuola ho dato alcuni accenni di storia e alcune informazioni sulla preside invece per l'indirizzo che frequento ho spiegato cos'è l'informatica e quali sono i principali metodi di programmazione usati per sviluppare il sito.

Informatica

L'informatica permette di risolvere dei problemi tramite una sequenza di operazioni che possono essere eseguite automaticamente da una macchina ( il computer ) su un insieme di dati in input, e rappresentare i risultati in output.

Metodi di programmazione usati per il sito:

  • Html è sostanzialmente un linguaggio di programmazione utilizzato per creare pagine e documenti su internet.
  • CSS è un linguaggio usato per definire la formattazione di documenti HTML, XHTML e XML, ad esempio i siti web e relative pagine web.
  • JavaScript® (spesso abbreviato con JS) è un linguaggio di programmazione, interpretato, orientato agli oggetti con first-class functions, ed è meglio conosciuto come linguaggio di scripting client-side per pagine web, nonostante questo è anche utilizzato in molte applicazioninon solamente basate sul web
